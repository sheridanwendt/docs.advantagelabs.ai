# GoDaddy Website Builder Access

Follow these steps to install the AEO Engine Pixel on your GoDaddy Website Builder site.

> **Note:** GoDaddy requires you to repeat this process for every page where the AEO Engine Pixel must run.

## Steps

**Step 1: Log in to your GoDaddy account.**

**Step 2: Go to Website Builder → Manage Site.**

**Step 3: Open the page where you want to add the script.**

**Step 4: Click Add Section → HTML.**

**Step 5: Paste the pixel script you copied from Search Atlas.**

**Step 6: Set the section height to 0 px.**

This hides the block visually while keeping it active.

**Step 7: Click Publish.**

**Step 8: Repeat these steps for all pages where the script should run.**
