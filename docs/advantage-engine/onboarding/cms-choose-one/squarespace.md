# Squarespace Access

Follow these steps to install the AEO Engine Pixel on your Squarespace site.

## Steps

**Step 1: Log in to your Squarespace dashboard.**

**Step 2: In the left menu, go to Settings → Advanced → Code Injection.**

**Step 3: Locate the Header field.**

**Step 4: Paste your AEO Engine Pixel `<script>` exactly as provided.**

**Step 5: Click Save in the upper-left corner.**

This places the Pixel in the site-wide `<head>` and ensures AEO Engine can detect it across all pages.
