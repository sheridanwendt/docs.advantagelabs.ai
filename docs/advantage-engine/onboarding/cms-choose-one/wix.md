# Wix Access

Follow these steps to install the AEO Engine Pixel on your Wix site.

## Steps

**Step 1: Log in to your Wix Dashboard in a new tab.**

**Step 2: Go to Settings > Custom Code.**

**Step 3: Click "+ Add Custom Code".**

**Step 4: In the window that opens, configure the following:**

- Set **Placement** to: **Head**
- Paste your AEO Engine Pixel `<script>` into the "Add your code here" box
- Set **Cookie Policy** to: **Essential**

**Step 5: Click Apply or Save.**

This applies the code site-wide, not just to a single page.
