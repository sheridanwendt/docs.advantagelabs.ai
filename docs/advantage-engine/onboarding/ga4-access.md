# How to Grant Access to Google Analytics 4 (GA4)

**Time required:** 2–3 minutes  
**Access level needed:** Editor

1. Go to [analytics.google.com](https://analytics.google.com)  
2. Select the correct GA4 property.  
3. Click **Admin** (gear icon in bottom-left).  
4. Under **Property Access Management**, click **+ → Add users**.  
5. Enter `AEO@AdvantageLabs.ai`.  
6. Set role to **Editor**.  
7. Click **Add**.  

✅ Access is granted immediately.
