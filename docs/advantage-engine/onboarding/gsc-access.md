# How to Grant Access to Google Search Console (GSC)

**Time required:** 2 minutes  
**Access level needed:** Full User

1. Go to [search.google.com/search-console](https://search.google.com/search-console)  
2. Select the correct website property (domain or URL prefix).  
3. Click **Settings** (left sidebar).  
4. Select **Users and permissions**.  
5. Click **Add user**.  
6. Enter `AEO@AdvantageLabs.ai`.  
7. Set permission to **Full**.  
8. Click **Add**.
